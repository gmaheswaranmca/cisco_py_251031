        crud ops wrt todos 
            - read all todos    GET /todos              array of todo js objects
            - read todo by id   GET /todos/12           one todo js object 
            - create todo       POST /todos             request : new todo js object -> response : created todo js object 
                request / json / body :
                {
                    "title" : "python api consuming learning",
                    "completed" : false,
                    "userId" : 1
                }
                response : status = 201 CREATED 
                {
                "title": "python api consuming learning",
                "completed": false,
                "userId": 1,
                "id": 201
                }
            - update todo     PUT   /todos/2        request : updatable todo obj -> response : updated todo js obj 
            req : {
                "userId": 1,
                "id": 2,
                "title": "quis ut nam facilis et officia qui",
                "completed": true
            }
            res: status = 200 OK 
            {
            "userId": 1,
            "id": 2,
            "title": "quis ut nam facilis et officia qui",
            "completed": true
            }
            - delete todo       DELETE /todos/2     todo will be deleted 
        
        REST API resource management - http based web service 
            GET  == read | query resource(s)
                - read all 
                - read by id 
            POST == create resource 
                request body will take one todo js object 
                response we will get the created todo js object  
            PUT == update resource 
                request body will take one edited todo js object 
                response we will get the updated todo js object 
            DELETE == delete the resource 
                for given resource id, the todo will be deleted 
                res : status = 200 OK 
            ---
            PATCH - partial update 
            ---
            all others will be used by browser and server
